Hauptfunktionen

Mehrsprachige Unterstützung

Die App unterstützt Deutsch, Französisch und Englisch, um eine breite Nutzerbasis anzusprechen.

Umfassende Spielregeln

Klar strukturierte und detaillierte Spielregeln, die den Spielern helfen, das Spiel schnell zu erlernen und zu meistern.

Benutzerfreundliches Interface

Intuitives Design mit SwiftUI, das eine nahtlose Benutzererfahrung gewährleistet.

Installation

  1. Systemanforderungen:
    • macOS mit der neuesten Version von Xcode.
    • Swift 5.5 oder höher.
  2. Repository klonen: 
    git clone https://github.com/dein-benutzername/yas-app.git
  3. Projekt öffnen:

    Öffne die YasApp.xcodeproj Datei mit Xcode.

  4. Abhängigkeiten installieren:

    Falls du externe Bibliotheken verwendest, installiere diese über Swift Package Manager oder CocoaPods.

  5. App ausführen:

    Wähle das gewünschte Zielgerät aus und klicke auf "Run".
